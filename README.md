# Территория игр

Этот репозиторий содержит исходный код интернет-магазина настольных игр, созданного на базе Django.

## Возможности
- Каталог товаров с детальными страницами
- Регистрация и авторизация пользователей
- Роли пользователей (администратор, менеджер, покупатель)
- Управление товарами и заказами
- Корзина и оформление заказов
- Панель аналитики на основе Matplotlib

## Быстрый старт
1. Создайте и активируйте виртуальное окружение:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Выполните миграции базы данных:
   ```bash
   python manage.py migrate
   ```
4. Запустите сервер разработки:
   ```bash
   python manage.py runserver
   ```
5. Откройте браузер и перейдите по адресу `http://localhost:8000/`.

## Запуск тестов
Для запуска встроенных тестов выполните:
```bash
python manage.py test
```
